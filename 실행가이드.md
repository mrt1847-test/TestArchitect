# TestArchitect 실행 가이드

## 🚀 빠른 시작

Electron 앱은 **서버가 필요 없습니다**. 로컬에서 바로 실행할 수 있습니다!

## 📋 실행 전 준비사항

### 1. 필수 프로그램 설치 확인

- **Node.js** (v16 이상) - [다운로드](https://nodejs.org/)
- **Python 3.x** - [다운로드](https://www.python.org/downloads/)

### 2. 의존성 설치

프로젝트 디렉토리에서 다음 명령어 실행:

```bash
npm install
```

이 명령어는 `package.json`에 정의된 Electron 및 기타 의존성을 설치합니다.

## ▶️ 실행 방법

### 방법 1: 일반 실행

```bash
npm start
```

### 방법 2: 개발 모드 실행 (DevTools 포함)

```bash
npm run dev
```

개발 모드에서는 자동으로 개발자 도구가 열려 디버깅이 가능합니다.

## 🧪 테스트 실행

1. 앱이 실행되면 메인 화면이 표시됩니다.
2. "테스트 스크립트 선택" 드롭다운에서 `example_test.py`를 선택합니다.
3. "실행" 버튼을 클릭합니다.
4. 결과가 "실행 결과" 패널에 표시됩니다.

## ⚠️ 문제 해결

### Python을 찾을 수 없다는 오류가 발생하는 경우

Windows에서 Python이 설치되어 있지만 인식되지 않는 경우:

1. Python이 PATH에 등록되어 있는지 확인
2. 명령 프롬프트에서 `python --version` 또는 `python3 --version` 실행하여 확인
3. 필요시 Python 재설치 시 "Add Python to PATH" 옵션 선택

### Electron이 실행되지 않는 경우

1. `node_modules` 폴더가 있는지 확인
2. 없으면 `npm install` 다시 실행
3. `package.json`의 `main` 경로가 올바른지 확인 (`src/main/main.js`)

### 스크립트 목록이 비어있는 경우

1. `scripts/` 디렉토리가 존재하는지 확인
2. `scripts/example_test.py` 파일이 있는지 확인
3. 파일 확장자가 `.py`인지 확인

## 📝 참고사항

- **서버 불필요**: Electron 앱은 독립 실행형 데스크톱 애플리케이션입니다.
- **로컬 실행**: 모든 처리는 로컬 컴퓨터에서 이루어집니다.
- **인터넷 불필요**: 기본 기능은 인터넷 연결 없이도 작동합니다.
